## 基础知识
* 字符串的find方法
```
str.find(sub[, start[, end]]) 
在切片s[start:end]中查找子字符串，如果被找到，返回最小的索引值，反之返回-1。可选的参数start和end是切片的标记。
```
* Ex.1
```python
str1="Harrdy2018"
sub="dy"
print(str1.find(sub))
print(str1.find(sub,2))
print(str1.find(sub,5,-1))
>>>4
4
-1
```
